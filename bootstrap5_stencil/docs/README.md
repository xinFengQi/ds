# 介绍

这是一个基于bootstrap5的标准组件项目，用于在文档构造中及平时一些demo项目中使用，属于ds个人系列的标准组件及工具库;

## 依赖

1. 使用Bootstrap5为基础ui库
2. 使用stencilJs为基础框架

## 标准组件库


### 使用方式

1. 引入js及css
2. 直接以标签形式使用
