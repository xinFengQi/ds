# chrome插件

## 功能
1. 重写书签页，实现书签页的多电脑同步功能
2. 实现脚本快捷执行功能，应用场景：快捷登录
3. 快捷便签功能
4. 本chrome插件的管理后台
5. gitee仓库文件管理功能
6. 博客管理功能(未开发完全)
